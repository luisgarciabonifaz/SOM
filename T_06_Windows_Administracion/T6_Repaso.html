Actividades de Repaso

Pon a prueba tus conocimientos sobre la administración y optimización de Windows.

Parte 1: Cuentas de Usuario y Grupos Locales

1.1. ¿Qué tipo de cuenta de usuario local en Windows tiene control total sobre el sistema y puede instalar programas o cambiar configuraciones que afecten a otros usuarios?

1.2. Para una gestión avanzada de usuarios, grupos y para establecer contraseñas o asignar pertenencias a grupos, ¿qué herramienta de Windows es más recomendable?

Parte 2: Directivas de Contraseñas

2.1. ¿Cuál de las siguientes NO es una configuración clave de las directivas de contraseñas en Windows?

2.2. Según las recomendaciones actuales, ¿cuál es la longitud mínima de contraseña que se considera buena práctica?

Parte 3: Directivas de Bloqueo de Cuentas

3.1. ¿Qué directiva define cuántos intentos de inicio de sesión fallidos se permiten antes de que una cuenta se bloquee temporalmente?

3.2. Si una cuenta se bloquea por intentos fallidos, ¿qué configuración determina el tiempo que permanecerá bloqueada?

Parte 4: Administración de Discos

4.1. ¿Cuál de las siguientes acciones NO se puede realizar con el Administrador de discos (diskmgmt.msc)?

4.2. ¿Qué tipo de volumen dinámico combina el espacio de varios discos para crear una única unidad lógica, distribuyendo los datos en "franjas" de manera simultánea para aumentar la velocidad?

Parte 5: Procesos y Servicios del Sistema

5.1. Si un programa no responde, ¿qué herramienta puedes usar para detenerlo forzosamente y qué pestaña te muestra los procesos en ejecución?

5.2. ¿Qué tipo de inicio de servicio hace que un servicio se inicie automáticamente con Windows, pero con un ligero retraso para mejorar el tiempo de arranque del sistema?

Parte 6: Optimización de Dispositivos de Almacenamiento

6.1. ¿Es beneficioso desfragmentar una unidad de estado sólido (SSD) para mejorar su rendimiento?

6.2. ¿Qué operación realiza el "Optimizador de unidades" de Windows en los SSDs para ayudar a mantener su rendimiento y alargar su vida útil?

Parte 7: Recursos Compartibles y Permisos

7.1. Si un usuario tiene "Control Total" en los permisos de compartición, pero solo "Lectura" en los permisos NTFS para una carpeta compartida, ¿cuál será su acceso efectivo?

7.2. ¿Qué comando de línea de comandos te permite ver la configuración de red de tu equipo, incluyendo la dirección IP y la máscara de subred?
